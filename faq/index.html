FAQ

  • "Divide By Zero or Overflow Error" error
    FIXED Use updated DSO client package.

  • "Error 42"
    This error occurs when the time in DOSBox differs from the real-world time by more than ±1 minute. Please ensure that the time on your host machine is correct and synchronized with the actual current time to avoid this issue.

  • mdark.exe - Main DSO client executable (DSO\resources\drives\c\DSO\mdark.exe)
    Always use the latest version of mdark.exe, as it includes important client-side fixes and updates. To get the latest version, message @crt0

  • sv.cfg - DSO main config file (DSO\resources\drives\c\DSO\sv.cfg)
    The format of the sv.cfg file is as follows:
    The first line must be empty (don't delete it!)
    The second line should contain the server address (use dso-status to get actual address)
    The third line should contain the server port (use dso-status to get actual port)
    The fourth line should contain your login!password


localhost
12345
login!pass

  • "Is there a specific server we should be pointed at for testing? Or both are equal valid to redirect us where we need to go?" - @Scoots
    All addresses point to the same DSO server, so just choose the one that is faster for your location.

  • Server adresses and ports To get the current server adressess and ports, use command dso-status in #server-status channel.

  • Sound To enable sound, press the Space key at the DSO loading screen, and make sure to choose either the wave or mss16 sound driver. By default, the volume is muted. To increase the volume, go to in-game settings using the Alt+P shortcut and adjust the level.

  • Animations By default, animations are off. To enable animations, press F6 in-game, or toggle it in Alt + P in-game settings.

  • Basic hotkeys
    F1 - Get help on keys in game
    F2 - List players in region
    F3 - List all players on the server
    Shift - Skip the game intro
    1..9 - Select answers in scripted dialogues with NPCs or during game events
    S,C,E - In the main menu for quick access: Start game, Create character, Exit
    Ctrl + F7 - (DOSBox) Start/Stop recording video output to DSO\capture\
